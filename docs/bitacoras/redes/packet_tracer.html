Usando Windows

Definiciones clave

Packet Tracer es un programa de simulacion de red para experimentar con diseños de red y los comportamientos de los mismos. Este programa ofrece lo siguiente:

  • Simulación
  • Visualización
  • Personalización
  • Evaluación
  • Colaboración
  • Facilita el aprendizaje y la enseñanza de conceptos tecnológicos complejos
  • El programa nos permite hacer pruebas en ambientes controlados
  • Podemos conocer el comportamiento de una actual o futura red sin comprar el equipo para experimentar con eso

Palabras clave

  • LAN: Local Area Network: Es la Red de Area Local que tiene un grupo de equipos que pertenecen a una misma organización, estos estan conectados dentro de un area geografica pequeña a traves de una red
  • SWITCH: Es un dispositivo de red que sirve para tener equipos interconectados dentro de una misma red, para formar una LAN
  • ROUTER [Enrutador]: Este tambien es un dispositivo de red que permite tener interconextados varios computadores o redes, mediante el enrutamiento de paquetes de datos

Espacios de trabajo

Packet tracer usa el espacio de trabajo logico y fisico

  • Logico: cada uno construye la topologia logica de su red, sin tener en cuenta escalas fisicas o limitaciones de construcciones, este tiene dos modos:
    1. Modo Tiempo Real: El simulador actua en tiempo real y no vemos como actuan los paquetes, solo se vera el resultado final
    2. Modo Simulacion: si vemos paso a paso como se comporta la red y como se mueven los paquetes a traves de la red y como va todo paso a paso
  • Fisico: cada uno coloca sus dispositivos de red en el lugar donde lo necesiten, ya sea edificios, locales, ciudades... se tiene que tener en cuenta como se afecta la red por la distancia o longitud del cableado

Registro y descarga

Podemos ingresar a la página oficial de CISCO para descargar el programa PACKET TRACER. Sin embargo es importante tener en cuenta que se debe iniciar sesión para poder descargar los recursos [para ello hay que crear una cuenta ]
